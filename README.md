# rest-api-with-mongoose
 <p>GET /accounts</p>
 <p>POST /accounts</p>
 <p>PUT /accounts/:id</p>
 <p>DELETE /accounts/:id</p>

# Usage
start the server with <strong><i>npm start</i></strong> with default port of 3000.

# Installation
<strong><i>npm install</i></strong>
